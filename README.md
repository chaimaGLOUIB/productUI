### Tech Stack

- React
- TypeScript
- Axios

### Run locally

npm install
npm run dev

### Features

- Responsive Product Cards
- API integration
- Category filtering
- Cart using React Context

### Sample Postman Request

GET:http://localhost:3000/products

### Layout Approach

This project uses a flexible grid layout to display products in a responsive grid. Components are structured using React functional components and styled with CSS to ensure maintainability and clarity.

### Responsiveness Considerations

The layout adapts to different screen sizes:

Mobile: single-column layout for easy scrolling

Tablet: two-column layout

Desktop: three to four-column layout for better product visibility
